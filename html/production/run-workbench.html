Starting milk volume

Initial volume is how much milk this run starts with. The system uses it to track losses, net yield, and packaging estimates. It should match the milk already planned / issued for this run — not a second guess.

Why this step exists

  • Warehouse issues milk via requisition (e.g. 50 L).
  • That amount becomes the run’s starting volume for yield tracking.
  • You should not re-type it unless the real milk on the floor is different.
  • Losses (spillage, evaporation) are recorded later and subtracted from this start volume.
