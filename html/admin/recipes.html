Production Recipes

Each recipe defines how to produce a base liquid (e.g. Chocolate Milk). The resulting liquid is then packaged into different bottle sizes (SKUs) in the Products catalog.

How the Production Flow Works

1 This Recipe page defines ingredients + process for making the liquid (e.g. 100L Fresh Milk vat) → 2 Production uses the recipe to create a batch → 3 Workers fill different bottle sizes from that batch (managed under Products → SKUs).

Never create separate recipes for "Fresh Milk 1L" and "Fresh Milk 500ml" — one recipe creates the liquid; the bottle size is a packaging decision.
